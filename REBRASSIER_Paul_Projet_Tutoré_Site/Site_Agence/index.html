Les prestations

L'agence propose la création de sites web personnalisés pour des acteurs publics ou privés. Elle s'occupe du front-end, mais également du back-end. Pour cela, le HTML, le CSS, Javascript, PHP et MySQL sont utilisés

Les réalisations

Nous sommes en train de réaliser un site web pour une formation à distance. Ce site nommé "Notes Universelles", doté de trois accès (étudiant, enseignant et direction) permet aux étudiants de consulter leurs notes, que les professeurs auront renseignées.
